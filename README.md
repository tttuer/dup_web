# 환경설정

## 윈도우

```
corepack enable
corepack prepare pnpm@latest --activate

pnpm install
```

## 맥

```
brew install pnpm
pnpm install
```

### arm64 에러 발생시

```
pnpm install @rollup/rollup-linux-x64-gnu --save-optional

```


# 실행

## 개발

```
pnpm dev
```
